# Interactive Dashboard with Python – Taipy

Sales Dashboard built-in Python and the Taipy library to visualize Excel data.

## Run the app
```Powershell
# vanilla terminal
python main.py

# quit
ctrl-c
```

## Demo
Sales Dashboard: https://dashboard.taipy.cloud/
